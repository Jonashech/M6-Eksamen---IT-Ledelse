## Om projektet

YUUMI er en statisk hjemmeside for en fiktiv japansk sushi-restaurant i Aarhus. Hjemmesiden er udviklet som vores eksamensprojekt hvor fokus har været at lære at arbejde struktureret med Git og GitHub i et team.

Selve produktet er bevidst holdt simpelt med ingen backend og minimalistiske sider. Det centrale læringsmål er at demonstrere god brug af versionsstyring. Den primære mening med projektet er hvordan man opretter branches, committer løbende, pusher og puller og samler arbejdet via pull requests. !!! VI skal nok lige have tilføjet noget her
